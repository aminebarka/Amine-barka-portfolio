# Camp Abdelmoula
## Client — B Studio

Site web pour un camp de luxe au cœur du Sahara tunisien à Tembaine, Douz. 51 tentes dont 27 royales. Design cinématique inspiré du désert avec palette sable/or/nuit. Cross-branding avec Défi du Désert Voyages.

**WordPress**
**PHP Custom**
**ACF**
**VIP Domaine**

Features:
- Hero plein écran avec étoiles CSS animées + dunes SVG
- Stats bar avec counter animation
- Section crosslink Défi du Désert avec countdown Raid J/H/M/S
- Menu slide-in avec sous-groupe Circuits & Raids
- Formulaire réservation multi-étapes + Google Maps GPS Tembaine

This project showcases my ability to create cinematically styled, custom WordPress websites.
