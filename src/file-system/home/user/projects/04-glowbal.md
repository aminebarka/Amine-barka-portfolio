# CloudCost Optimizer
## Personal Initiative

An AI-powered multi-cloud SaaS platform aggregating cost and usage data from AWS, Azure, and GCP with intelligent optimization recommendations.

**Frontend:** React
**Backend & API:** Python, FastAPI
**Base de données:** PostgreSQL
**ML & Data:** Scikit-learn
**DevOps & IaC:** Terraform (Multi-cloud)
**Cloud:** AWS (Lambda, autres services)

This project demonstrates my expertise in cloud financial management, AI/ML integration, and multi-cloud architecture.
