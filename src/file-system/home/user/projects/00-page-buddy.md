# DevOpsFlow
## Graduation Project | Neopolis Development

A comprehensive Multi-Cloud Deployment Automation SaaS platform for multi-environment (Dev, Staging, Prod) deployment automation.

**Frontend:** React.js
**Backend & API:** Django REST Framework (Python)
**Base de donnees:** PostgreSQL
**DevOps & Cloud:** Docker, Kubernetes (Helm), Terraform, GitLab CI, Jenkins
**Monitoring:** Prometheus, Grafana
**Message Broker:** Kafka
**Providers Cloud:** AWS, Azure, GCP, DigitalOcean, OVHcloud

This project demonstrates my expertise in DevOps, cloud architecture, and infrastructure automation at scale.
